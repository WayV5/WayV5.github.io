Git的一些基本使用命令

发表于 2023-06-01 更新于 2023-06-04 分类于 开发工具
本文字数： 295 阅读时长 ≈ 1 分钟

第一次 从远端拉取代码

git clone 远端地址

上班改代码之前第一件事情 先拉取代码（更新）

git pull

修改自己代码

自己改代码，此时项目在本地的文件系统中有变化了。

提交过程

查看修改了什么

git diff

对修改/添加了的文件，在git版本库中进行更改

git add . #点表示所有文件

git add abc.txt

对删除了的文件，在git版本库中进行处理

git rm efg.txt

创建一个提交

git commit -m ‘name’ #”起个commit名字”

再一次pull

git pull

push到远端

git push -u origin main

一次push能处理多个commit

回滚commit

git reset
